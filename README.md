# Mutils
A tool for managing Mudae collections and tracking kakera earnings.

## Collection
Import your collection using `$mmy+k=rlz+dx+i-s` in Discord. Search, sort, export, edit key counts and notes. Key types are color-coded: Bronze (1-2), Silver (3-5), Gold (6-9), Chaos (10+).

## Roll Calculator
Calculate wish and starwish probabilities based on your server settings. Configure pool size, badges, tower perks, and $bw roll investment. Includes optimizer to find the best split between rolls and boostwish. Save and export configurations.

## Kakera Statistics
Log kakera claims manually or bulk import from Discord logs. View totals, averages, and trends. Charts show daily income, type distribution, cumulative growth, and day-of-week activity. Advanced stats include rolling averages, momentum, volatility, threshold calculator, and linear growth projections.

## Optimizer
Get suggestions for which characters to enable or disable.

## Tech
React, TypeScript, .NET 10, PostgreSQL, Discord OAuth
